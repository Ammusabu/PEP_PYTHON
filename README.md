# PEP Python Codes

## Overview
This repository contains Python programs and practice codes written during the **PEP (Professional Enhancement Program) classes conducted by Lovely Professional University (LPU)** for placement training.

I selected the **Python Full Stack track under CyberScholars**, and this repository includes the Python exercises, practice problems, and coding examples covered during the PEP training sessions.

## Purpose
The purpose of this repository is to:
- Practice Python programming concepts
- Store codes written during PEP classes
- Improve problem-solving skills for placements
- Maintain a record of training exercises

## Topics Covered
Some of the Python concepts practiced include:
- Variables and Data Types  
- Conditional Statements  
- Loops  
- Functions  
- Lists, Tuples, Sets, and Dictionaries  
- Basic Problem Solving
